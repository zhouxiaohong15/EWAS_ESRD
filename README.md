# EWAS_ESRD

We analyzed EPIC methylation array data using R. The process involved several key steps: first, we performed quality control and normalization, followed by batch effect correction. Next, we conducted epigenome-wide association studies (EWAS) on a large sample to identify CpG sites associated with end-stage renal disease (ESRD). Finally, we applied Bacon correction, a Bayesian-based method designed to estimate and correct for bias and inflation in test statistics.
